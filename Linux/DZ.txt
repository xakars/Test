1) Посмотреть где я - pwd
2) Создать папку - mkdir testQA
3) Зайти в папку - cd testQA
4) Создать 3 папки - mkdir testQA1 testQA2 testQA3
5) Зайти в любоую папку - cd testQA2
6) Создать 5 файлов (3 txt, 2 json) - touch QAtxt1.txt QAtxt2.txt QAtxt3.txt QAjson1.json QAjson2.json 
7) Создать 3 папки - mkdir QAtest1 QAtest2 QAtest3
8. Вывести список содержимого папки - ls
9) + Открыть любой txt файл - vim QAtxt1.txt
10) + написать туда что-нибудь, любой текст - жмем i(режим ввода), пишем текст
11) + сохранить и выйти - :wq
12) Выйти из папки на уровень выше - cd ..

17) вывести несколько первых строк из текстового файла - head -3 QAtxt.txt
18) вывести несколько последних строк из текстового файла - tail -3 QAtxt.txt
